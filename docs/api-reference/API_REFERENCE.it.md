# Riferimento API

Questo documento descrive la direzione della API pubblica di openings.dev.

La API espone lo stesso indice normalizzato usato dalla web app per abilitare integrazioni, automazioni e analisi su prodotti terzi.

## Stato

La API e pubblica ed evolutiva. Alcuni contratti possono cambiare mentre filtri e regole di normalizzazione maturano.

## Obiettivi di design

- Schema coerente tra repository sorgente.
- Paginazione prevedibile per sincronizzazione incrementale.
- Identificatori stabili per deduplicazione.
- Campi chiari per stack, seniority, remoto e localizzazione.
